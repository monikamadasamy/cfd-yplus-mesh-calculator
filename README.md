# cfd-yplus-mesh-calculator
Interactive CFD y⁺ &amp; near-wall mesh calculator that estimates Reynolds number, friction velocity, wall shear stress, and first cell height from basic flow inputs. It also recommends turbulence models and helps configure accurate boundary-layer mesh settings for ANSYS Fluent and OpenFOAM.
CFD Near-Wall Mesh Calculator is a lightweight, browser-based pre-processing tool that helps students, researchers, and entry-level engineers determine accurate near-wall mesh parameters for boundary layer resolution. The first cell height near a wall is one of the most critical meshing decisions in any CFD simulation — it directly controls whether turbulence models and boundary layer physics are resolved correctly. Calculating it manually requires multiple intermediate steps, empirical correlations, and careful unit handling, which frequently leads to errors. This tool removes that friction entirely.

The user inputs free-stream velocity, fluid density, dynamic viscosity, characteristic length or hydraulic diameter, and a target y⁺ value. From these, the calculator computes Reynolds number and flow regime, skin friction coefficient via Prandtl or Blasius correlations, wall shear stress, friction velocity, and first cell height in both metres and millimetres — ready to paste directly into ANSYS Meshing or OpenFOAM. Boundary layer thickness, recommended growth rate, and suggested inflation layer count are also provided. Turbulence model guidance — k-ω SST, LES, or k-ε with wall functions — is automatically selected based on the computed y⁺ range. Intelligent warnings flag laminar flow conditions, compressibility effects, and mesh inconsistencies.
Built as a single HTML file with no dependencies, it runs instantly in any browser and deploys via GitHub Pages.
